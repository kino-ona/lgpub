3,400W 초고화력

센 불은 요리 시간을 단축하는 것은 물론 더 맛있는 요리를 만드는데 반드시 필요한 조건이죠.

화력이 강하면 라면을 끓이더라도 더 꼬들꼬들한 면발로 더 맛있게 즐길 수 있답니다.

최대 3,400W의 초고화력이라면 집에서 만드는 요리도 전문 레스토랑 못지않은 맛있는 요리로
재탄생하게 될 거예요.


· 인덕션 3,400W 우측 대화구 터보 기준 (단독 동작 시)

· 3구 동시 사용 가능하며, 2개 이상 화구 동시 동작 시 제품 최대 출력 범위(3,400W) 내에서
화구별 출력을 자동으로 조정합니다. (터보 모드 사용 시 다른 화구 사용 불가함)

· 3,400W 초고화력 : 한국 에너지관리공단 홈페이지 기재된
*동급 인덕션 제품의 **단일화구 최대 출력 기준(2022년 1월)
*동급 인덕션 : 해당 자료 내 1) 국산 제조, 2) 인덕션 3구 이상의 화구 구성 플러그인 제품에 한함.
**단일화구 최대 화력 : 각 제조사 홈페이지 및 사용설명서에 기재된 출력에 근거

총 출력, 최고 출력 모두 3,400W의 최고 화력으로

가스레인지보다 최대 2.6배 빠르게 요리가 가능하죠.


· 자사가 제시한 시험 방법으로 자사 전기레인지(우측 대화구, 3,400W)/가스레인지(3,400W) 대상,
물 최초 온도(25℃)로부터 70℃ 상승한 95℃ 도달까지의 소요시간 측정

· 실험 조건 : 물 1L, 21cm 냄비(바닥부), 3,400W 최대 점화 / 터보 기준

· 실험 모델 : BEI3MQT(인덕션), HB622AB(가스레인지)

· 자사 시험 결과를 3자 시험 기관인 UL코리아가 검증한 결과(2021.12)로
가스레인지 대비 2.6배 빠른 것으로 확인함

· 실험 결과는 실제 사용 환경에 따라 다를 수 있음

· 22년 1월 출시 이후 3구 인덕션 모델에 한 함. (BEI3*QT/ BEI3*QT*/BEI3**QT/BEI3**QT*)

쿼드 인버터

3구 인덕션 2022년도 신모델에 탑재된 쿼드 인버터 기술은 전류가 통행하는 입구를 2배로 늘려
특히 가열이 잘 안되었던 저효율 인덕션 용기에 에너지를 온전히 전달해 화력 손실은 줄이고
가열 성능을 개선했어요.

이제 저효율 인덕션 용기를 사용하더라도 답답함 없이 기존 자사 모델 대비
최대 2.3배 빠른 조리가 가능하죠.


· 쿼드 인버터는 우측 대화구에만 적용되어 있음

· 자사가 제시한 시험 방법으로 자사 전기레인지(우측 대화구, 3,400 W)/자사 기존 전기레인지(3,300 W) 대상, 물 최초 온도(25℃)로부터 70℃ 상승한 95℃ 도달까지의 소요시간 측정

· 실험 조건 : 물 1.5 L, 선정된 인덕션 전용 저효율 용기(스펙 아래), 실험 모델 별 우측 대화구 최대 점화
(터보 모드) 기준

· 실험 모델 : BEI3MQT, BEI3MST

· 자사 시험 결과를 3자 시험 기관인 UL코리아가 검증한 결과(2021.12)로 BEI3MQT가 BEI3MST 대비
2.3배(약 57%) 빠른 것을 확인함

· 물 끓임 실험 용기명 : 키친플라워 브랜드 쿠킨 IH 인덕션 양수냄비 20cm (바닥부 실측 16.5 cm)

· 인덕션 전용 저효율 용기는 BEI3MST, BEIMQT 모델로 용기 가열 지수를 확인하여 용기 가열 지수가
'5~8'사이의 저효율 용기를 사용함 인덕션 전용 저효율 용기는 인덕션 접촉면이 19~22 cm의 바닥면이 평평한 원형 냄비를 사용 권장함(바닥면의 자석이 붙는 면적이 19~22 cm 사이)

· 인덕션 사용 가능 용기의 바닥 소재는 자석이 붙는 소재임

· 인덕션 전용이어도 제조사 및 용기마다 가열 성능이 다를 수 있음

· 이해를 돕기 위해 연출된 이미지입니다.

· 용기가열지수는 소비자 이해를 돕기 위해 LG전자가 도입한 내부 기준으로, 10에 가까울수록
용기가 잘 가열됩니다.
0~4 : 사용을 권장하지 않는 용기, 5~8 : 가열효율이 낮은 저효율 용기, 9~10 : 가열 효율이 높은 고효율 용기

· 용기가열지수가 낮고 용기가 가벼울수록 소음이 커질 수 있으나, 이는 가열 효율을 높이기 위한 정상 동작이니 안심하고 사용하세요.

· 2022년 1월 출시 이후 3구 인덕션 모델에 한 함. (BEI3*QT/ BEI3*QT*/BEI3**QT/BEI3**QT*)
