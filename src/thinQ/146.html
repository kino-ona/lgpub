실내 오염물질을 배출하는 환기는 미세먼지 차단 못지않게 중요해요.

LG 주거용 환기 시스템은 깨끗한 공기 유입과 생활 속 유해 물질의 배출은 물론, 냉·난방비 절약에도 도움을 주는 청정 환기 시스템이죠.

01

에너지 회수

창문을 열어 환기하면 오염된 공기와 함께 냉·난방 에너지까지 빠져나가죠. 하지만 환기 시스템은 오염된 공기만 배출되고 고효율의 전열 교환기가 바깥으로 버려지는 냉방 에너지 60%, 난방 에너지 74%를 회수하여 에너지가 절약되고, 그만큼 냉·난방비도 절약되죠. 특히 여름철 에어컨을 가동할 때 창문 환기 대비하여 전기료를 크게 줄일 수 있어요.

에너지 회수 효과

냉·난방비 절감 효과

한국표준규격(KS B 6879)에 준하여 당사에서 자체 시뮬레이션 한 결과입니다.


[시뮬레이션 조건]
· 실내·외 온도 및 습도
- 냉방 : [실외] 온도 35℃, 상대습도 40%

- 난방 : [실외] 온도 2℃, 상대습도 74%

· 전용면적 85㎡, 법정 환기량 0.5회/hr, 에어컨 EER 6.0, 가스보일러 효율 90%, 도시가스 가격 14,23/MJ, 전기를 350kWh/월 사용하는 가구 기준
· 필터 교체 등 유지 보수 비용은 별도

02

쾌속청정

인공지능이 휘센 공기청정 에어컨과 환기의 최적 운전 조건을 찾아 최대 2배 이상 빠르게 미세먼지를 제거해 주죠.

공기청정키트가 있는 휘센 시스템 에어컨과 환기 연결 시 적용 가능해요.

미세먼지 제거 속도 시험 결과

· 시험 기간 : 2019년 12월 11일 ~ 13일
· 시험 장소 : 서울 잠실 파크리오 아파트
· 시험 제품 : 주거용 환기(150CMH), 1Way 공기청정 카세트 TT샤시
· 시험 조건 : 세대 내 상온 상습 조건
· 사용 분진 : 대기진(외기 미세먼지 농도 PM2.5 약 50㎍/㎥에서 시험 실시)
· 시험 방법 : 각 운전 조건 별 실내(거실) 미세먼지 농도 변화량을 측정하여 비교.
· 시험 결과 : 공기청정 에어컨과 환기 연동 사용 시 미세먼지 농도가 50㎍/㎥에서 35㎍/㎥ (공기질 보통 수준)으로 감소하는데 약 14분 소요되며, 환기 제품 단독 사용 시 동일 조건으로 미세먼지가 감소하는데 약 30분 소요됨.

03

심야 외기 냉방 운전

실외 온도가 실내 온도보다 낮아지는 밤 시간대에는 환기가 자동으로 켜져요.


이때 더운 실내 공기는 밖으로 배출하고 반대로 시원한 바깥공기는 실내로 공급해 줘요.


이후 실내 온도가 적정 온도에 도달하면 자동으로 환기가 꺼진답니다.


이렇게 쾌적한 실내 환경을 만들어줄 뿐만 아니라 에어컨 사용 시간을 절감시켜 에너지를 절약하는 효과가 있죠.

04

계절 맞춤 운전

실내/외 온도를 감지하여 계절에 따라 최적 운전 모드로 자동 전환하여 에너지를 절약해요.

05

환기량 맞춤 운전

CO2 센서가 실내 CO2 농도를 감지하고 오염도 변화에 따라 풍량을 스스로 조절해 실내를 쾌적하게 유지해요.

· 유선 리모컨을 통해 CO2 농도와 상태 모니터링이 가능합니다.
· 환기량 맞춤 운전 기능은 옵션으로 선택 가능한 CO2 센서가 부착된 제품에 한해 지원합니다.

01

프리필터 자외선 살균

UVnano 기술이 적용되어 공기가 가장 먼저 닿는 프리필터의 세균 및 바이러스 번식을 99.99% 막아주죠.

· 시험 일자 : 2021년 8월
· 시험 모델 : Z-H0150B2SR
· 시험 환경 : 25±2℃, 50±10% RH 2시간 제품 동작 후 측정
· 시험 균주 : 황색포도상구균, 표피포도상구균, 폐렴막대균, 박테리어파지
· UVnano 살균은 UV LED 빛에 닿는 면에 한하며, 제품 수명에 따른 광충력 감소 등 실사용 환경에 따라 다를 수 있습니다.

02

공기 통로 항균·항곰팡이

전열교환기와 유로에 적용된 항균, 항곰팡이 소재로 곰팡이가 자랄 수 없게 해 항곰팡이 최고 등급인 0등급1)을 확보했어요.

1) 곰팡이 저항성 등급 기준 (ASTM G21-15)
0 = 자라지 못함
1 = 10 % 이하로 자라남
2 = 10~30 % 자라남
3 = 30~60 % 자라남
4 = 60 % 이상 자라남

열 교환 소자 곰팡이 저항 시험
· 시험 일시 : 2020년 11월
· 시험 기관 : FITI시험연구원
· 시험 규격 : ASTM G21-15
· 시험 균주 : Aspergillus brasiliensis, Chaetomium globasum, Penicillium funiculosum, Trichoderma virens, Aureobasidium pullulans
· 시험 조건 : 28~30℃, 85%RH 이상, 28일
· 시험 결과 : 자라지 못함 (0등급)
· 실험실 측정 기준으로 실 사용 환경에서는 달라질 수 있습니다.
실험 결과는 제품 초기 성능을 측정한 결과이며 사용 시간에 따라 달라질 수 있습니다.

유로(EPS) 항균 시험
· 시험 일시 : 2020년 8월
· 시험 규격 : JIS Z 2810: 2010, 필름 밀착법
· 시험 방법 : 시험균액을 35±1℃, 90%RH에서 24시간 정치 배양 후 균수 측정
· 시험 결과 : 항균 활성치 R4.6(균주1), R6.2(균주2)
· 실험 결과는 제품 초기 성능을 측정한 결과이며 사용 시간에 따라 달라질 수 있습니다.

· 시험 균주 : Aspergillus niger ATCC 9642, Chaetomium globosum ATCC 6205, Pencillium pinophilum ATCC 11797, Glicladum virens ATCC 9645, Aureobasidium pulluants ATCC 15233, Cladosporium Cladosporioides IFO 6348

03

HEPA H13 등급 미세먼지 필터

HEPA H13 등급 미세먼지 필터 적용으로 0.3㎛ 미세먼지를 99.97% 차단해 걱정없어요.

HEPA H13 등급의 미세먼지 필터 · 당사의 퓨리케어 360° 공기청정기와 동일한 HEPA 필터 원단을 사용
· 시험 일시 : 2021년 1월
· 시험 기관 : 한국건설생활환경시험연구원
· 시험 대상 : AS351NNFA 모델 기준
· 시험 조건 : 0.01㎛ 직경 입자의 제거효율
· 시험 방법 : 실험 환경 새 필터 기준 300㎥ 챔버, 20분 이상 가동
· 시험 환경 조건 및 시험 방법 : SPS-KACA 00-132:2018 규격 Modified 진행, 공기청정기 ‘클린부스터모드 터보풍’ 운전에 따른 0.01㎛ 직경의 먼지 제거율 측정(40분간)
· 시험 결과 : 0.01㎛ 미세먼지 99.999% 제거 (27분 이상 가동 시 99.999% 제거)
· 제조사 측정 기준 필터 가공 크기 : 기존 필터 26,32㎛, 플러스 필터 18.25㎛
· 시험 결과로 입자 크기 등 시험 조건과 사용 환경에 따라 차이가 있을 수 있습니다.
